# DatesToCronConverter
## **Краткое описание**
Переводит даты формата yyyy-MM-dd'T’HH:mm:ss в формат cron.
Пример: 
Input:
	"2022-01-25T08:00:00"
	"2022-01-25T08:30:00"
	"2022-01-25T09:00:00"
	"2022-01-25T09:30:00"
	"2022-01-26T08:00:00"
	"2022-01-26T08:30:00"
	"2022-01-26T09:00:00"
	"2022-01-26T09:30:00"

	Output:
	"0 0/30 8-9 * * *"
